# x402 Mantle Facilitator Data Availability

## Objective
The original research framework identified the "Questflow SANTA multichain facilitator volume" on Mantle as a potential metric for measuring x402 activity. The goal was to find the official Mantle contract address for the x402 facilitator and track its transaction volume.

## Investigation Steps
1. **Searched official documentation**: Reviewed Questflow's documentation (questflow.ai) and Mantle's official x402 gateway announcements.
2. **Searched GitHub & block explorers**: Searched GitHub codebases and Mantlescan for verified contracts matching "SANTA", "Questflow", or "x402 Facilitator" on Mantle (Chain ID 5000).
3. **General Web Search**: Conducted broad searches for "Questflow SANTA multichain facilitator contract address Mantle".

## Findings
There is no publicly listed, verified contract address for a Questflow SANTA multichain facilitator on the Mantle network in available documentation.

While Questflow’s S.A.N.T.A and x402 Facilitator integrations on Mantle are announced and discussed, the specific contract addresses appear to be managed internally through proprietary platform infrastructure rather than published as a single public address.

## Conclusion
Due to the lack of a reliable, attributable on-chain source or verified contract address, it is impossible to accurately track this metric directly from the blockchain at this time. We will not fabricate or estimate this data.

**This missing data is a finding itself**: For researchers measuring agent economies, the opacity or proprietary nature of specific protocol addresses (unlike standard ERC-8004 registries) presents a significant barrier to open analytics. This highlights a limitation in current cross-chain x402 transparency.
